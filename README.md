# miguel-acai

Projeto simples de gerenciamento de clientes e pedidos baseado da empresa Miguel Açaí de Ampére - PR, feito com autorização da mesma.

## Requisitos

- Node.js
- Backend [click aqui](https://github.com/devHonorio/miguel-frontend.git)

## Instalação

- [Linux](#linux)
- [Windows](#windows)
- [macOS](#macos)

### Linux

1. Instale o Node.js usando o gerenciador de pacotes da sua distribuição ou [aqui](https://nodejs.org/).

### Windows

1. Instale o Node.js através do instalador disponível [aqui](https://nodejs.org/).

### macOS

1. Instale o Node.js usando o Homebrew:

```bash
brew install node
```

## Repositório

```bash
git clone https://github.com/devHonorio/miguel-frontend.git
cd miguel-backend
```

Para rodar o projeto, siga os passos abaixo:

1. Instale as dependências do projeto:

```bash
npm i
```

2. Em seguida, inicie o servidor de desenvolvimento:

```bash
npm run dev
```

3. Abra [http://localhost:3000](http://localhost:3000) no seu navegador para ver o resultado.
